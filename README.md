# hubot-symphony-scripts

Some example Symphony scripts for hubot

## Installation

In hubot project repo, run:

`npm install hubot-symphony-scripts --save`

Then add **hubot-symphony-scripts** to your `external-scripts.json`:

```json
[
  "hubot-symphony-scripts"
]
```
